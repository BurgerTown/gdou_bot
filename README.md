# GDOU_BOT

用于 `@GDOU_Water` telegram群组中的Bot

## 上手指南

以下指南将帮助你在本地机器上安装和运行该项目

### 安装要求

1. Python3.7
2. [pip](https://pypi.org/project/pip/)
3. git

## 拥有功能

* `/jw` -- GDOU教务系统的地址
* `/yjpj`  -- GDOU计算机协会的一键评教软件下载地址
* `/weather_now`  -- 获取当前GDOU的天气
* 每天 00:00 定时推送GDOU当天天气预报
